Proyecto de Programación Numérica en GNU Octave

Estructura:
- main.m: Script principal con menú interactivo para ejecutar los algoritmos.
- algoritmos/: Carpeta con funciones modulares para interpolación y cuadratura.
    - newton_interp.m: Interpolación de Newton.
    - lagrange_interp.m: Interpolación de Lagrange.
    - trapecio_simple.m: Regla del trapecio (simple).
    - trapecio_multiple.m: Regla del trapecio (múltiples segmentos).
    - simpson_13_simple.m: Simpson 1/3 simple.
    - simpson_38_simple.m: Simpson 3/8 simple.
    - simpson_13_multiple.m: Simpson 1/3 múltiple.
    - simpson_mixto.m: Simpson mixto (par/impar).

Cómo ejecutar:
1. Abrir GNU Octave y situarse en la carpeta del proyecto.
2. Ejecutar el script principal:
   >> main
3. Seguir las instrucciones del menú para probar cada algoritmo.
4. Los ejemplos incluidos muestran el uso típico y generan gráficas cuando es relevante.

Todos los algoritmos están documentados en español y utilizan nombres de variables claros.
